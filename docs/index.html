CI/CD ¿en local?

Git Hooks

CI/CD en local: Husky

Permite configurar Hooks de Git rápidamente:

Instalación:


npm install husky --save-dev

Configuración:


// package.json
{
   "husky": {
      "hooks": {
         "pre-commit": "npm test",
         "pre-push": "npm test",
         "...": "..."
      }
   }
}
